# Synthesizer, in progress

**Features**:
  * Two oscillator, sub oscillator and LFO
    * 4 different waveforms (sin, triangle, sawtooth and square)
    * Detune
  * Filter with lowpass, highpass and bandpass modes
  * Adjustable cutoff and resonance
  * ADSR
  * Noise-generator
  
**Depedencies**: 
  * JUCE
